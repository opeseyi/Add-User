# ADD USERS

TTHis is parctice project to add user.
Built with **React**

## To use

1. Clone
   ```
   git clone https://github.com/opeseyi/Add-User
   ```
2. Install dependencies

   ```
   npm install
   ```

3. Run frontend
   ```
   npm start
   ```

### This should be up and runing in your browser and you should be able to interact with it
